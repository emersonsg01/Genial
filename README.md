# Genial — Calculadora e Ferramentas

Projeto interativo com módulos para cálculos matemáticos, física, finanças, conversões e gráficos.

**Para instruções completas de setup, testes e uso, veja [SETUP.md](SETUP.md).**

## Quick Start

```bash
npm install
npm run test -- --run
npm run dev
```
